6.1 Getting Started

The first step is to correctly read your data into the software:

  1. If you haven’t already, install RStudio;
  2. Download the data. If it is an Excel (.xlsx) file, open it in Excel and save as CSV;
  3. Open RStudio and create a new R markdown file. Save this file to the same folder as the data!
  4. Go to Session > Set Working Directory > To Source File Location. R now knows where to locate your data;
  5. Create a new code chunk and run the following:
grasshoppers <- read.csv("grasshoppers.csv")

From here on out, just read the text, run the code in R, adjust the R code if needed, and answer the questions.

Note: For the sake of privacy, this page uses an old version of the data with scrambled names. I have also not linked the actual data. Please ask an assistant if you are unable to find it.

6.3 Creating a Subset

We just read the whole data set. It contains the data from all groups Let’s make a subset of the measurements performed by group A:

Subset <- grasshoppers[grasshoppers$Group == "A", ]

What does this code do?

  • Subset <-: Create a variable named Subset;
  • grasshoppers[x, y]: From the data set grasshoppers, select only rows x and columns y. If y is left blank, it selects all columns;10
  • grasshoppers$Groep == "A": From the data set grasshoppers, select the variable Group and check if it is equal to the value A (TRUE) or not (FALSE).

6.4 Comparing Means with a \(T\)-Test

Below is an example of a \(t\)-test, using some standard data in R:

t.test(extra ~ group, data = sleep, alternative = "less", var.equal = TRUE)
## 
##  Two Sample t-test
## 
## data:  extra by group
## t = -1.8608, df = 18, p-value = 0.03959
## alternative hypothesis: true difference in means between group 1 and group 2 is less than 0
## 95 percent confidence interval:
##        -Inf -0.1076222
## sample estimates:
## mean in group 1 mean in group 2 
##            0.75            2.33

This particular \(t\)-test is a one-sided \(t\)-test for equal variance. The output shows that the mean of group 1 is \(0.75\) hours of extra sleep, while the mean of group 2 is \(2.33\). The \(p\)-value for this test is \(0.0396\). If we use a level of significance of \(\alpha = 0.05\), then group 2 is has a significantly larger mean than group 1, because \(0.0396 < 0.05\).
